AUTONOMOUSc Edge Node Installer

Plug in, approve this machine, and let the runtime finish the rest.

This app is the normal owner path for checkout builds. Give this machine a name, review what we inferred locally, and keep this window open while first run checks Docker, validates the NVIDIA runtime, verifies Hugging Face access, pulls the runtime image, fills the local model cache, warms the startup model, and finishes the browser approval flow automatically.

1. Checking Docker

Quick Start makes sure Docker Desktop and local disk space are ready first.

2. Checking NVIDIA runtime

The installer verifies that a supported NVIDIA runtime is available and chooses deterministic presets from local VRAM.

3. Validating HF access

If the startup model is gated, Quick Start validates the saved Hugging Face token before any large download begins.

4. Pulling image

The runtime image is cached locally first so retries and later restarts are faster.

5. Downloading model

Quick Start shows model-cache progress while the first local download fills in the background.

6. Warming model

Once the files are local, the runtime finishes the warm-up and confirms the startup model is actually responding.

7. Claiming node

The installer opens a browser approval flow, shows the claim code, and keeps polling until approval lands.

8. Node live

The node agent and supporting services come online and the dashboard switches to a live-ready state.
