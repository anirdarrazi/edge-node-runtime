AUTONOMOUSc Local Node Service

One local app for Quick Start, runtime health, updates, and support.

This app is the normal owner path for checkout builds. Start with Quick Start, approve the machine once, and then use the same local app to keep the node healthy and online, even through the slower first-time download and warm-up steps.
